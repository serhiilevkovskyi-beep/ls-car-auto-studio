# LS-CAR Auto Studio Website

Готовый сайт на React + Vite.

## Как запустить локально

1. Установите Node.js: https://nodejs.org/
2. Откройте папку проекта в терминале.
3. Выполните:

```bash
npm install
npm run dev
```

После этого откройте ссылку, которую покажет терминал, обычно `http://localhost:5173`.

## Как загрузить на Vercel

1. Зайдите на https://vercel.com
2. Создайте новый проект.
3. Загрузите эту папку или подключите GitHub-репозиторий.
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Нажмите Deploy.
